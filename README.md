# Memojo Landing Page

A modern, minimalistic landing page for Memojo, featuring their innovative MemojoFind product that allows users to search their photo galleries using natural language.

## Features

- Responsive, mobile-friendly design
- Modern Apple-inspired minimalist interface
- Interactive elements and animations
- Sign-up forms for Android and iOS beta testing
- Privacy-focused messaging

## Project Structure

```
LandingPage/
├── css/
│   └── styles.css
├── js/
│   └── script.js
├── index.html
└── README.md
```

## Getting Started

1. Clone or download this repository
2. Open `index.html` in a web browser to view the landing page

## Customization

- **Colors**: Modify the CSS variables in the `:root` selector in `styles.css`
- **Content**: Edit the text in `index.html`
- **Animations**: Adjust the animations in `script.js`
- **Images**: Add your own images to enhance the visual appeal

## Browser Compatibility

This landing page is designed to work in all modern browsers including:
- Chrome
- Firefox
- Safari
- Edge

## Technology

The landing page uses plain HTML, CSS, and JavaScript without any dependencies, making it lightweight and fast to load.
